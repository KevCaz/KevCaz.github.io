One more scope in spell-check

September 26, 2018
  Debian Atom spell-check

I often use R markdown within Atom. By default, spell-check does not check .Rmd file and so I needed to add the corresponding scope to the list of grammars checked. It was a two steps process:

  1. find the scope with the command: Log cursor scope (use the command-palette);

  2. copy-paste the scope and add it to the list of scopes that are already checked.
