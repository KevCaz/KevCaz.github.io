From Lunar Lobster to Mantic Minotaur

November 5, 2023
  Ubuntu release installation

I have recently decided to upgrade to the most recent Ubuntu release on my Dell Precision machine, in other words, I wanted to get closer to Ubuntu’s release cycle. Previously, I would reinstall the latest long time release (LTS) every time a new update pops up (see the documentation). About 3 months ago, I switched to Ubuntu 23.04 (Lunar Lobster) and yesterday I upgraded to the October’s release, Mantic Minotaur (23.10). The two upgrades were seamless, I just had to change a setting in the update-manager (see this) and confirm the upgrade when I got notified. Once the installation completed, I had to edit a couple of source files for which only the latest LTS is available (22.04, a.k.a Jammy Jellyfish). Here is my current configuration in a nutshell:
